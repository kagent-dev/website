# Kagent Custom Resource Definitions (CRDs)

## What are CRDs?

Custom Resource Definitions (CRDs) in Kubernetes provide a mechanism to extend the Kubernetes API by defining new, custom resource types. These custom resources behave similarly to built-in Kubernetes objects like **Pods** or **Deployments**, but are tailored to specific application or organizational needs.

---

### Installing Kagent CRDs

> ⚠️ Skip this step if already done in **Quick Start**.

```bash
kagent install
```
To access the list of CRDs:
```bash
kubectl get crd -n kagent
```
