# Channels

Channels connect your kagent agents to the messaging platforms your team already uses. Instead of switching to a separate UI, your team interacts with agents directly inside Slack threads, Telegram chats, Discord servers, or any MCP-compatible client.

Every channel uses the [A2A (Agent-to-Agent) protocol](/docs/kagent/examples/a2a-agents) under the hood. This means any agent you build works on any channel — no per-platform agent code required.

## How channels work

1. A lightweight bot or connector runs alongside your cluster
2. Incoming messages are forwarded to kagent agents via A2A
3. Agents process the request using their configured tools and LLM
4. Responses flow back through the same channel to the user

Text messaging is supported on every channel. Rich formatting, media attachments, interactive elements, and Human-in-the-Loop approval buttons vary by platform.

## Platform notes

| Channel | Auth method | Threading | Media | HITL buttons |
|---------|------------|-----------|-------|-------------|
| **Slack** | Bot token (Bolt SDK) | Native threads | Images, files | Slack actions |
| **Discord** | Bot token (Discord API) | Thread replies | Embeds, attachments | Reactions |
| **Telegram** | Bot token (polling) | Reply chains | Images, GIF, video | Inline keyboards |
| **WhatsApp** | Business API | Reply quotes | Images, documents | Quick replies |
| **MCP Clients** | Direct endpoint | Session-based | N/A | Native prompts |

## Delivery notes

- **Slack** multi-person DMs route as group chats. Group policy, mention behavior, and session rules apply to MPIM conversations.
- **Telegram** replies that contain markdown image syntax are converted into media replies when the bot supports it. The bot uses long-polling from inside the cluster — no public endpoints required.
- **Discord** bots connect via the Discord gateway. Slash commands and message-based interactions are both supported.
- **MCP clients** connect directly to agent endpoints. No bot middleware is needed — the client speaks MCP natively.
