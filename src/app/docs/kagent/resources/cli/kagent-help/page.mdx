# `kagent help`

Help provides help for any command in the `kagent` CLI.

```bash
kagent help [command] [flags]
```

**Global Flags:**
- `--config` - config file
- `--kagent-url` - KAgent URL (default "http://localhost:8083")
- `--timeout` - Timeout (default 5m0s)
