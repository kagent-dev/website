# Agent CRD Reference

Complete reference for the kagent Agent Custom Resource Definition (v1alpha2).

## Agent Resource

```yaml
apiVersion: kagent.dev/v1alpha2
kind: Agent
metadata:
  name: <agent-name>
  namespace: <namespace>
spec:
  type: Declarative | BYO
  declarative: <DeclarativeAgentSpec>
  byo: <BYOAgentSpec>
  skills: <SkillForAgent>
  allowedNamespaces: [<string>]
```

## DeclarativeAgentSpec

```yaml
declarative:
  runtime: python | go
  modelConfig: <ModelConfig-name>
  systemMessage: <string>
  promptTemplate:
    dataSources:
      - configMapRef:
          name: <ConfigMap-name>
  tools:
    - type: McpServer | Agent
      mcpServer: <McpServerTool>
      agent: <AgentTool>
  a2aServer:
    enabled: <bool>
    port: <int>
  memory:
    enabled: <bool>
    embeddingProvider:
      type: <string>
      model: <string>
      secretRef:
        name: <Secret-name>
  context:
    eventsCompaction:
      enabled: <bool>
  deployment: <SharedDeploymentSpec>
```

## McpServerTool

```yaml
mcpServer:
  name: <string>
  kind: RemoteMCPServer | MCPServer
  toolNames:
    - <tool-name>
  requireApproval:
    - <tool-name>
  propagateHeaders:
    enabled: <bool>
```

## AgentTool

```yaml
agent:
  name: <string>
  namespace: <string>
```

## SkillForAgent

```yaml
skills:
  refs:
    - image: <registry/repo:tag>
  gitRefs:
    - url: <git-url>
      ref: <branch/tag/commit>
      path: <subdirectory>
      name: <custom-name>
  gitAuthSecretRef:
    name: <Secret-name>
  insecureSkipVerify: <bool>
```

```yaml

  name: <name>
```

```yaml
  protocol: SSE | STREAMABLE_HTTP
  url: <string>

    - secretRef:
        name: <Secret-name>
      key: <header-name>
```
